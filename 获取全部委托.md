# /open/api/all_order-获取全部委托 <br/>
#### /open/api/all_order (get)：已成交和已取消
| 参数 | 填写类型 | 说明 |
| ----- | :----: | ----: |
|symbol |必填|市场标记，btcusdt，详情看下面|
|pageSize|选填|页面大小|
|page|选填|页码|
|api_key|必填|api_key|
|time|必填|时间戳|
|sign|必填|签名|

#### 返回
| 字段 | 实例 | 解释 |
| ----- | :----: | ----: |
|code |0||
|msg|"suc"|code>0失败|
|data|如下:||
